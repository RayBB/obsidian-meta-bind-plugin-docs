The `minValue` argument lets specify a minimum value for a <InputFieldLink type={InputFieldType.SLIDER}></InputFieldLink> or <InputFieldLink type={InputFieldType.PROGRESS_BAR}></InputFieldLink> input.
The value passed to this argument must be a number.

### Allowed Input Fields

<InputFieldArgumentAllowedInputFields type={InputFieldArgumentType.MIN_VALUE}></InputFieldArgumentAllowedInputFields>

### Examples

```
INPUT[slider(minValue(-10)):bind_target]
```

```
INPUT[slider(minValue(-10), maxValue(10)):bind_target]
```
